Next.js 15 + React 19 + TypeScript 풀스택 개발자. 프로덕션용 스타터킷을 직접 구축하고 있으며 shadcn/ui, Tailwind CSS v4, React Hook Form + Zod, usehooks-ts 등 현대적 스택에 익숙함.

코드 리뷰 피드백은 한국어로 요청하며, 코드 주석과 커밋 메시지도 한국어 선호. 변수명/함수명은 영어 유지.

기술 수준: 고급 (직접 설계한 아키텍처 패턴, 접근성 고려, 성능 최적화 패턴 적용 중)
